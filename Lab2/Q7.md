Q7: Implement an algorithm for making a simple calculator with all the operators (+,-,*,/,%)

```
START
INPUT Num1, Num2
PRINT "What operation do you want to do 'add', 'subtract', 'divide', 'multiply', 'mod'"
INPUT OP
IF OP == 'add'
  Than Result = Num1 + Num2
ELSEIF OP == 'subtract'
  Than Result = Num1 - Num2
ELSEIF OP == 'divide'
  Than Result = Num1 / Num2
ELSEIF OP == 'multiply'
  Than Result = Num1 * Num2
ELSEIF OP == 'mod'
  Than Result = Num1 % Num2
PRINT Result
```
